# Roadmap tecnico

## Etapa 0

Cerrar decisiones comerciales que impactan codigo:

- planes,
- politica de pago vencido,
- metodo de cobro,
- politica de activacion.

## Etapa 1. Fundaciones

- Crear app Next.js con App Router.
- Configurar TypeScript estricto, ESLint, Prettier, Vitest y Playwright.
- Integrar Supabase Auth.
- Definir `profiles`, `admin_users`, `access_requests`.
- Implementar layout base, middleware y manejo de sesion.
