# Flujos criticos

## Alta y activacion

1. La persona crea `access_request`.
2. Admin revisa y aprueba o rechaza.
3. Admin asigna plan o habilitacion comercial.
4. La cuenta pasa a `active`.
5. La usuaria completa onboarding.

## Flujo de fabricante

1. Crear unidades.
2. Crear recursos.
3. Registrar compras de recursos.
4. Crear productos fabricados.
5. Crear receta.
6. Registrar produccion.
7. Verificar descuento de recursos.
8. Verificar aumento de stock de producto.
9. Registrar venta.
10. Verificar descuento de producto.
11. Verificar ingreso economico y dashboard.

## Flujo de reventa

1. Crear productos de reventa.
2. Registrar compra de productos.
3. Verificar aumento de stock.
4. Registrar venta.
5. Verificar descuento de stock e ingreso.

## Flujo de reversion

1. Abrir documento origen.
2. Validar impacto historico y stock actual.
3. Generar movimientos compensatorios.
4. Reaplicar nuevos movimientos si corresponde.
5. Mantener consistencia de inventario y finanzas.

## Flujo administrativo

1. Revisar solicitudes.
2. Aprobar o rechazar.
3. Asignar plan.
4. Registrar pago.
5. Bloquear o desbloquear cuenta.
6. Consultar auditoria.
